def app.linker (url, parentid=-1)
 Goes to the website with the given url, extracts some information about it, stores them in the database and recursively goes all the website links contained in this website. The extracted information about websites include unique address id, website url, parent website id, crawling date and time, number of links in the website and how many of these links are inserted into the database.
